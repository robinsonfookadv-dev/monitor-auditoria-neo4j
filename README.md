# 🛡️ Monitor de Auditoria com Neo4j e Python

Projeto focado em **Integridade de Dados** e **Cadeia de Custódia Digital**, integrando conceitos de Direito Digital (LGPD) com tecnologias de Cibersegurança.

### 🚀 Funcionalidades
* **Hashing SHA-256:** Geração de assinaturas únicas para garantir que os dados não foram alterados.
* **Grafo de Relacionamentos:** Modelagem no **Neo4j Aura** para visualizar conexões entre usuários, dados sensíveis e vulnerabilidades.
* **Automação em Python:** Script para alimentação do banco de dados e monitoramento.

### 🛠️ Tecnologias
* **Linguagem:** Python 3.x
* **Banco de Dados:** Neo4j (Grafos)
* **Segurança:** Hash SHA-256 para integridade.
